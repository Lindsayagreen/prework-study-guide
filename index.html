HTML

  • HTML is used to create structure of webpage
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • Metadata is data about our webpage located in head element
  • Semantic HTML element that describes its content & role played on webpage

CSS

  • Cascasding Style Sheets
  • adds style to page
  • inline CSS: style an element directly in the HTML file
  • internal style sheet: CSS rules for the webpage in the head section of the HTML file
  • external CSS style sheet: most common way to style a webpage, allows developers to keep all their CSS rules in a seperate file
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • allows web devlopers to collaborate on projects without overwriting each other's work
  • allows devlopers to revert to previous versions of the code if a problem is discovered

JS

  • core web language
  • a high-level dynamic programming that allows users to interact with websittes
  • core feature: variable is simply a named container that allows us to store data in our code
  • Each line of JavaScript code ends with a semicolon
  • code is executed from top to bottom. However, using conditional statements such as if statements allows us to interrupt control flow.
  • array is a single variable that is used to hold a group of data
